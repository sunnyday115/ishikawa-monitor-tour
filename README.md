# 石川県モニターツアーWebサイト

![石川県モニターツアー](hero-bg.jpg)

## 概要

石川県観光連盟が運営する、石川県内のモニターツアーを紹介するWebサイトです。補助金が適用された特別価格でツアーに参加できます。

## 特徴

✨ **美しいデザイン** - 石川県の伝統色（金箔の金、加賀友禅の深紅、日本海の青）を基調  
✨ **モダンなUI** - グラスモーフィズム、グラデーション、マイクロアニメーション  
✨ **レスポンシブ** - モバイル、タブレット、デスクトップ対応  
✨ **インタラクティブ** - スクロールアニメーション、FAQアコーディオン、パララックス効果

## 技術スタック

- HTML5
- CSS3（カスタムプロパティ、グリッドレイアウト、フレックスボックス）
- バニラJavaScript（Intersection Observer API使用）
- Google Fonts（Noto Sans JP、Shippori Mincho）

## ローカル実行

```bash
# HTTPサーバーで起動
npx http-server ./ -p 8080 -o
```

ブラウザで `http://localhost:8080` にアクセス

## 主要セクション

1. **ヒーローセクション** - 圧倒的なビジュアルで訴求
2. **モニターツアー概要** - 3つの特徴をカード形式で表示
3. **利用の流れ** - 5ステップのタイムライン
4. **ツアーピックアップ** - 6つの魅力的なツアー
5. **よくある質問** - アコーディオン形式FAQ
6. **フッター** - 連絡先情報とリンク
